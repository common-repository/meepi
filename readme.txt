A free tool that builds email lists, increases social shares, gathers feedback, runs competitions all within your own site. Can be triggered by exit intent, on-load or scrolling actions, you decide. 

== Description ==

Meepi.com provide free tools that allow you to easily add Meepi.com code to your wordpress install. 

The average website bounce rate is 41 - 55%. That’s over half of your visitors leaving without ever engaging with your content. Crazy right?! 

= Special Features =

1. **Monetise** - Retain visitors and increase revenue by reacting to your visitors’ exit intent; incentivise them to make purchases before abandoning you and convert fleeting visitors into valuable customers.
2. **Capture Emails** - Collect data from your visitors, build up your mailing lists and generate leads for sales and email campaigns.
3. **User Feedback** - Find out what really drives your customers digital behaviour and what they expect from you as a Company so that you can plan your marketing strategies for the future. 
4. **Social** - Increase your social interaction without Facebook and Twitter tools.
5. **Sweepstakes** - Use our sweepstake and competition tools to run giveaways and get vital user engagement.
6. **Custom HTML** - Create your own unique overlays for any type of marketing activity you like; advertise new products, special offers and events.


== Installation ==

1. Upload `Meepi.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Retrieve your Meepi code from meepi.com.
4. Enter your Meepi code into the 'Meepi' Tab in Wordpress admin and set status to 'enabled'.
4. You're set.

== Frequently Asked Questions ==

= Is Meepi free?=
Yes there will always be a free tier.

= Do I need to know how to code?=
No, there are graphical interfaces to customise each Meepi.

= Will the plugin slow my site down?=
No, the code loads asynchronously, meaning it will load in the background and not slow your page loading.

= Can I customise the overlays?= 
Yes, each Meepi is configurable and if you are feeling confident you can create your own unique HTML or even inlcude a feed you have access to.

= Can I export the emails and Campaigns?=
Yes, at any time you can export all data and emails collected. By law we are not allowed to market to them, these emails are all yours.

= Can I integrate with email providers?=
Yes, there are integrations being added all the time. We have Mailchimp and Aweber sorted for example.

= Do I need to create an account on Meepi.com?=
Yes, this is easy though. Simply login and create your campaigns. Then copy and past the code you will be given within Meepi.com.
